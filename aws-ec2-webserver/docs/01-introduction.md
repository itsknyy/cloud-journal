# 1. Description

This project demonstrates how to deploy a simple web server using `Amazon EC2`, configure network access using `Sercurity Groups`, install `Apache` and host a basic HTML webpage.

## a. Project Objectives

- Launch a Linux server using AWS EC2.
- Configure security rules(HTTP/SSH).
- Connect to the instance using SSH.
- Install Apache web server.
- Deploy a simple HTML webpage.
- Test the website over the internet.

## b. Technologies Used

- AWS EC2 (Ubuntu Server 22.04 LTS)
- Apache Web Server
- Linux CLI/SSH
- Security Groups
- HTML

## c. Prerequisities

- AWS Free-Tier Account
- Basic command-line knowledge
- SSH client (Windows, macOS, Linux)
- An internet browser
